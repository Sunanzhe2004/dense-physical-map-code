# Albedo Examples

This directory stores fixed albedo example groups for released main-experiment documentation and qualitative illustration.

Current source:

- `l3d124s8endidr6wiyui5ngmluf3p3wi888/004`
- `l3d124s8endidr6wiyui5ngmluf3p3wi888/009`
- `l3d124s8endidr6wiyui5ngmluf3p3wi888/016`

Each example folder currently includes:

- `<id>_im.png`: source RGB image
- `<id>_albedo.png`: aligned albedo target
- `<id>_mask.png`: valid-mask image
- `<id>_im.json`: per-image analysis metadata used for albedo prompting

These files are intended as fixed examples for the albedo main-experiment pipeline rather than as a standalone benchmark split.
